# MoeMail Logo 字样精修设计

## 目标

在已经完成的全站字体清爽化基础上，进一步把 `MoeMail` 品牌字样打磨得更精致克制：

1. 不更换品牌语言，不引入新的强个性展示字体。
2. 通过字重、字距、渐变、间距和 hover 节奏，让字样更像成熟产品 Logo。
3. 同时覆盖顶部导航 Logo 与品牌头部 Logo，保持统一视觉气质。

## 设计方向

采用“品牌字样单独设层”的精修方案：

1. 继续使用无衬线字体系，避免和正文的清爽产品感冲突。
2. 将 `MoeMail` 字样从宽松的 `tracking-wider` 调整为更克制的轻微负字距。
3. 降低渐变的跳跃感，让紫色过渡更柔和。
4. 保留像素信封图标作为品牌记忆点，文字负责稳定与质感。

## 影响范围

仅调整以下组件：

1. `app/components/ui/logo.tsx`
2. `app/components/ui/brand-header.tsx`

必要时允许补充一个本地预览 HTML，用于快速查看成品效果，但不改业务逻辑。

## 验证方式

1. 用文本断言确认新字样类名已落地。
2. 运行 `npm run lint` 与 `npm run build`，确认组件变更未破坏构建。
3. 启动本地预览服务，展示改后的导航 Logo 与品牌头部 Logo 效果。
